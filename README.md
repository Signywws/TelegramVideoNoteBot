CircleNoteBot

Telegram-бот для конвертации видео в формат «кружочков» (Video Note).
Реализован на чистом Go с прямыми HTTP-запросами к Telegram API – без сторонних обёрток.
Рассчитан на высокую конкурентность благодаря фиксированному Worker Pool и асинхронной обработке.

---

Возможности:
- Принимает видеосообщения (до 60 секунд) и преобразует их в квадратные Video Notes (разрешение 640×640).
- Мгновенная обратная связь – бот сразу отвечает пользователю, а обработка идёт в фоне.
- Конкурентность – пул воркеров фиксированного размера предотвращает перегрузку и обрабатывает запросы от множества пользователей одновременно.
- Сохранение данных – оригиналы видео и готовые кружочки сохраняются на диск, метаданные записываются в MongoDB.
- Graceful shutdown – при нажатии Ctrl+C бот завершает текущие задачи и только потом останавливается.
- Устойчивость к ошибкам – переживает сетевые сбои, некорректные форматы и слишком большие файлы без падений.

---

Aрхитектура
```text
Telegram API
      │
Worker Pool
      │
Long Polling (polling)
      │
Dispatcher
      │
Handlers (text, video, other)
      │
Worker Pool
      │
VideoProcessor (download → FFmpeg → upload)
      │
FileStorage + MongoDB
```

Уровень представления – клиент Telegram, поллер, диспетчер, обработчики.

Инфраструктура – Worker Pool для асинхронных задач.

Сервисный слой – обработка видео (FFmpeg), бизнес-логика.

Слой данных – файловое хранилище (диск) + MongoDB-репозиторий.

Технологический стек


Компонент	Выбор
```text
Язык	Go 1.21+
Telegram API	Прямые HTTP-запросы, Long Polling
Обработка видео	FFmpeg (через обёртку ffmpeg-go)
База данных	MongoDB (официальный драйвер mongo-driver)
Файловое хранилище	Локальный диск (интерфейс позволяет подключать любое хранилище)
Конкурентность	Собственный Worker Pool (горутины + канал)
Логирование	Стандартный log/slog
```

Необходимые компоненты
- Go версии 1.21 или выше
- FFmpeg, установленный и доступный в PATH
- MongoDB (локальный экземпляр или облачный сервис)
- Telegram токен бота (получить у @BotFather)

Быстрый старт

1. Клонирование и установка зависимостей
```bash
git clone https://github.com/yourusername/circlenotebot.git
cd circlenotebot
go mod download
```

2. Установка переменных окружения

Создайте файл .env в одной директории с main.go и задайте переменные окружения:
```bash
TOKEN = "ваш_токен"
POLLTIMEOUT = 30
MONGO_URI = "url_подключения_mongoDB"
```

3. Конфигурация
```text
Переменная окружения	- Значение по умолчанию	Описание
TELEGRAM_BOT_TOKEN	- обязательно	Токен Telegram Bot API
MONGO_URI	- mongodb://localhost:27017	Строка подключения к MongoDB
STORAGE_PATH - не обязательно ./media	Корневая папка для сохранения файлов
WORKER_POOL_SIZE - 4	Количество одновременно выполняемых задач по обработке видео
POLL_TIMEOUT - 30	Таймаут Long Poll в секундах
```

4. Сборка и запуск
```bash
go run ./cmd/bot
```

--- 

Ключевые проектные решения:
- Отсутствие сторонних библиотек для Telegram – только стандартный HTTP, что даёт полный контроль.
- Фиксированный пул горутин для обработки видео и Pull Request – предотвращает исчерпание ресурсов.
- Паттерн «обработчик-перехватчик» – чистое разделение ответственности.
- Файлы хранятся в папках пользователей (media/{userID}/{original|video_note}/{uuid}.mp4) – исключены коллизии.
- Коллекция MongoDB files хранит метаданные по каждому обработанному видео.
